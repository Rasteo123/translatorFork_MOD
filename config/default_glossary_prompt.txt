This output is not a mere list of translations; it is a **coherent system** where each decision logically reinforces the others, creating a unified and predictable lexical world.

**CRITICAL DISTINCTION:** 
The Translator LLM already possesses perfect knowledge of the source and target languages. It does NOT need a dictionary.

- **Your Output:** A system of **recorded decisions** that locks each **Lexical Anchor** (canonical terms, names, ranks) to a definitive translation.
