# GitHub Release Draft

Suggested tag: `v10.5.17`
Suggested title: `translatorFork_MOD v10.5.17`
Range: `v10.5.16..HEAD` (`2026-04-22`)

## Release Body

## Что нового

- Добавлены настраиваемые цвета темы интерфейса: можно отдельно управлять основными акцентами, а настройки сохраняются между запусками.
- В `translator-only` добавлен доступ к настройкам прокси прямо из окна переводчика, вместе с индикацией текущего статуса.
- В предпросмотре AI-исправлений глоссария включено inline-редактирование строк до применения правок.
- Логика auto-filter redirect отделена от repack flow, поэтому перенаправление и повторная упаковка теперь работают предсказуемее в авто-сценариях.

## Исправления

- Исправлено определение локальных моделей в `AI Consistency`, из-за чего local-like обработчики могли резолвиться неверно.
- Исправлен сброс proxy/session state после изменений конфигурации сетевых параметров.
- Исправлен сброс страницы виджета глоссария после автообновлений.
- Убраны лишние `resize`-срабатывания в виджете управления ключами.
- Добавлена очистка повторяющихся дублей в концовках глав при обработке EPUB.

## Технические изменения

- Доработан `build_translator_linux.sh` и минимальный набор зависимостей для translator-only Linux build.
- Добавлены новые регрессионные тесты для прокси, auto-filter redirect, AI correction preview, тем оформления, AI consistency, revalidation и очистки EPUB-дубликатов.
- Сохранение и применение пользовательских цветов темы подключено как в основном приложении, так и в translator-only входной точке.

## Важно

- Это следующий релиз после `v10.5.16`, опубликованного `21 April 2026`.
- Breaking changes для этого релиза не выделены.
- Linux-ассет `translatorFork-translator` по-прежнему соответствует минимальной translator-only сборке.
